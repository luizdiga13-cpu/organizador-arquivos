# 📂 Organizador Automático de Arquivos

Aplicativo desenvolvido em Python com interface gráfica usando Streamlit.

O sistema organiza automaticamente arquivos em categorias como:

- Imagens
- Vídeos
- PDFs
- Planilhas
- Músicas
- Documentos
- Arquivos Compactados
- Programas
- Arquivos Python

---

# 🚀 Funcionalidades

✅ Escolher pasta automaticamente  
✅ Organizar arquivos por extensão  
✅ Criar pastas automaticamente  
✅ Barra de progresso em tempo real  
✅ Logs de organização  
✅ Interface gráfica moderna  

---

# ▶ Como executar

## 1. Clone o projeto

```bash
git clone URL_DO_SEU_GITHUB
```

---

## 2. Entre na pasta

```bash
cd organizador_arquivos
```

---

## 3. Instale as dependências

```bash
pip install -r requirements.txt
```

---

## 4. Execute

```bash
streamlit run app.py
```

ou

```bash
python -m streamlit run app.py
```
